# Design System

## Typography

- **Headings:** `font-display font-bold text-2xl text-gov-navy` (Outfit)
- **Body:** Default sans (DM Sans)

## Colors (Tailwind)

- **gov-navy:** `#0f172a` — primary headings, nav active
- **gov-blue:** `#1d4ed8` — primary actions, links
- **gov-blue-light:** `#3b82f6` — hover
- **slate-50 to slate-800** — backgrounds and text

## Components

- **Page header:** `flex justify-between items-center mb-6`, h1 with design tokens above
- **Primary button:** `px-4 py-2 bg-gov-blue text-white rounded-lg text-sm font-medium`
- **Secondary button:** `px-4 py-2 bg-slate-200 rounded-lg text-sm` or `border border-gov-blue text-gov-blue`
- **Cards:** `bg-white rounded-xl shadow border border-slate-100 p-6`

## Consistency

All Pre-Bid and admin pages use the same header pattern and button styles. Auto-Builder, Manual, Evidence, and Appendices share the same layout and tokens.
